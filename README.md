# CanSat
En este repositorio se encuentran los archivos que corresponden a códigos, diagramas electrónicos y diseños 3D de los prototipos de CanSats diseñados como parte de un trabajo final de grado.

Institución: Facultad de Ingeniería de la Universidad Nacional de Asunción. Paraguay

Título del trabajo: Diseño e implemetación de un Picosatelite tipo CanSat

Autores: Lucas Moreira y José Moreira

Año: 2020
